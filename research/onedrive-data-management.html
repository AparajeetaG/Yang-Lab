Overview

This Python script scans a locally‑synced OneDrive Human folder (all subfolders), counts files by type (including DICOM .dcm/.ima, DAT .dat, NIfTI .nii/.nii.gz), and exports a multi‑sheet Excel report. It also marks each subject as Processed (if any .mat) or Not Processed (only .dat/.ima).

Prerequisites

  • Python 3.9+ (3.10+ recommended)
  • Packages: pandas, openpyxl
  • OneDrive signed in and the Human folder synced locally
pip install pandas openpyxl

Download

⬇ Download the Python Script

If the file opens in the browser, right‑click the button and choose “Save link as…”.

Run

Windows:

python human_folder_analysis.py --root "C:\Path\To\OneDrive\Human" --out "C:\Path\To\Output"

macOS/Linux:

python3 human_folder_analysis.py --root "/Users/<you>/Library/CloudStorage/OneDrive-.../Human" --out "$HOME/Documents"

The script prints a summary and saves a timestamped Excel file with sheets: Overview, File_Types, one per main subfolder, and All_Folders.
